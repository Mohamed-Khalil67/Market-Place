About Flask Marketplace

Flask Marketplace is a full-featured online marketplace built with Flask where users can register, login, buy, and sell items.

Features

  • User Authentication
    • Secure registration and login system
    • Password hashing with Werkzeug
    • Session management
  • Marketplace Functionality
    • Browse available items
    • Post items for sale
    • Purchase items from other users
    • User dashboard to manage listings
  • Shopping Cart
    • Add products to cart
    • Update quantities
    • Remove items
    • Checkout process
  • Payment Processing
    • Secure payment integration
    • Order history
    • Order status tracking
  • Database
    • SQLAlchemy ORM
    • SQLite (or PostgreSQL/MySQL ready)
    • Models for Users, Items, Transactions

Technologies Used

  • Backend: Python, Flask
  • Frontend: HTML5, CSS3, Bootstrap
  • Database: SQLAlchemy, SQLite
  • Authentication: Flask-Login
  • Forms: Flask-WTF
  • Security: Flask-Talisman, rate limiting
  • Performance: Flask-Caching, database indexing
